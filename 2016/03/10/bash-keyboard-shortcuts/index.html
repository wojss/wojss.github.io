Bash快捷键

发表于 2016-03-10   |  

移动

Ctrl + a 行首，Home键相同
Ctrl + e 行末，End键相同
Ctrl + p 上一条命令，↑键相同
Ctrl + n 下一条命令，↓键相同
Ctrl + b 光标向左移，←键相同
Ctrl + f 光标向右移，→键相同
Alt + b 光标向左跳一个词，光标位于词首，Ctrl + ←键相同
Alt + f 光标向右跳一个词，光标位于次末，Ctrl + ←键相同
Ctrl + xx 在行首/行末间跳转

编辑

Ctrl + l 清屏，clean命令相同
Ctrl + d 删除，Delete键相同
Ctrl + h 退格，Backspace键相同
Alt + d 从光标向右删除一个词
Ctrl + w 从光标向左删除一个词
Ctrl + u 从光标删除到行首
Ctrl + k 从光标删除到行末
Alt + t 光标前后的词位置交换
Ctrl + t 光标前后的字符位置交换
Ctrl + y 粘贴最后一次切割的内容
Alt + r 撤销改变并恢复到原历史记录一样
Tab 补全文件/目录的名字

历史

Ctrl + r 在历史记录中搜索最后一条匹配的命令
Ctrl + o 直接执行通过Ctrl + r搜索到的命令
Ctrl + g 退出搜索模式
!! 执行上一条命令
!abc 执行最后以abc开头的命令
!abc:p 打印最后以abc开头的命令
!$ 上一条命令最后一个参数
Alt + . 同!$
!* 上一条命令所有参数
